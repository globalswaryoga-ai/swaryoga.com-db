## Visual Design Details

### Colors Used
- **Primary Green**: #1b5e20, #2e7d32 (Buttons, headings)
- **Light Green Background**: #e8f5e9, #f1f8e9 (Testimonials, mentor section)
- **White**: Background
- **Dark Gray**: #222, #666 (Text)
- **Light Gray**: #f5f5f5 (Subtle backgrounds)

### Typography
- **Headings**: Bold, 28px-48px
- **Subheadings**: Bold, 20px-24px
- **Body Text**: Regular, 14px-18px
- **Links/Buttons**: Bold, 16px

### Spacing
- **Section padding**: 50px-80px vertical
- **Content max-width**: 1200px
- **Card gaps**: 20px-30px
- **Mobile padding**: 20px-30px

### Animations

## Mobile vs Desktop Layout

### Desktop View
- 2-column hero (text + image side-by-side)
- 3-column video grid (highlights)
- 4-column testimonials grid
- 4-column video testimonials grid
- Full-width sections with max-width containers

### Mobile View
- 1-column hero (stacked image below text)
- 1-column video grid (stacked)
- 2-column testimonials (in pairs)
- 1-column video testimonials (stacked)
- Full-width with padding on edges
