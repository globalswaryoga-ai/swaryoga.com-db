## üé® Customization Guide

### Lighten Colors
Add `bg-primary-100` or `bg-primary-200` for lighter variations

### Darken Colors
Add `bg-primary-800` or `bg-primary-900` for darker variations

### Adjust Opacity
Use Tailwind opacity: `bg-primary-500/50` for 50% opacity

### Create Gradients
Use `bg-gradient-eco` or `bg-gradient-premium` classes

### Add Shadows
Use `shadow-eco` or `shadow-eco-lg` for green-tinted shadows
