## Notes for Developers

### URL Parameters Explanation
```
https://images.unsplash.com/photo-ID?w=500&h=600&fit=crop
```
- `w=500` - Width in pixels
- `h=600` - Height in pixels (matches card dimensions)
- `fit=crop` - Crops image to exact dimensions
- This ensures consistency across all cards

### Loading Performance
All Unsplash URLs are CDN-optimized and cached globally for fast loading.

### Responsive Images
Consider using Next.js Image component with srcSet for better performance:
```typescript
<Image 
  src={workshop.image}
  alt={workshop.name}
  width={500}
  height={600}
  quality={85}
  priority={false}
/>
```
