## Key Features

| Feature | Description |
|---------|-------------|
| **Hero Section** | Title, description, details, image + register button |
| **Intro Video** | Full-width YouTube embed with controls |
| **Learning Points** | 5 customized bullet points per workshop |
| **Highlight Videos** | 3 videos in grid with play button overlay |
| **Video Modal** | Click video to open fullscreen modal |
| **Mentor Section** | Customized bio for each workshop |
| **Testimonials** | 4 text testimonials with green styling |
| **Video Testimonials** | 4 participant videos with names |
| **Blinking Buttons** | Animation effect on all register buttons |
| **Multiple CTAs** | 5+ registration buttons throughout page |
| **Fixed Button** | Sticky register button at bottom |
| **Responsive** | Works on mobile, tablet, desktop |
| **Professional Design** | Green theme, proper typography, spacing |

## Button Animations

All register buttons have a **blinking effect**:
```css
@keyframes blink {
  0%, 100% { opacity: 1; }
  50% { opacity: 0.7; }
}
.blink-btn {
  animation: blink 1.5s infinite;
}
```

---

## Video Players

### Embedded Videos
- YouTube embeds (no autoplay)
- Full controls (play, pause, fullscreen, captions)
- Responsive sizing
- Mobile-friendly

### Modal Videos
- Click on video thumbnail to open modal
- Fullscreen overlay
- Close button (X) on top right
- Click outside to close
- Modal opens full-width video player

## Data Structure

```typescript
interface WorkshopLandingData {
  heroImage: string;           // URL to hero image
  introVideoUrl: string;       // YouTube embed URL
  whatYouWillLearn: string[];  // 5 learning points
  highlightVideos: Array<{
    title: string;
    url: string;
  }>;                          // 3 videos with titles
  mentorInfo: string;          // Mentor biography
  testimonials: string[];      // 4 text testimonials
  videoTestimonials: Array<{
    name: string;
    url: string;
  }>;                          // 4 video testimonials
  finalCTA: string;            // Final call-to-action
}
```
