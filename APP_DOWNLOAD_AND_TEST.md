## ‚ö†Ô∏è If Installation Fails

### Error: "App not installed"

**Cause:** Old version still installed

**Fix:**
```
1. Settings ‚Üí Apps
2. Find "Swar Yoga"
3. Tap ‚Üí Uninstall
4. Restart phone
5. Try installing again
```

### Error: "Install unknown apps" permission

**Cause:** Phone doesn't allow installation from this source

**Fix:**
```
1. On phone, open: Settings ‚Üí Apps & Notifications
2. Find: "Files" (or whatever app you're using to install)
3. Tap ‚Üí Permissions
4. Toggle ON: "Install unknown apps"
5. Go back and try installing APK again
```

### Error: "Not enough storage"

**Cause:** Phone is full

**Fix:**
```
1. Delete old apps/photos to free up space
2. Restart phone
3. Try again
```

### APK won't download from Google Drive

**Cause:** Drive permissions

**Fix:**
```
1. On phone, open: Google Drive app (not web)
2. Right-click file ‚Üí Download
3. Open Files app
4. Go to Downloads folder
5. Tap APK file to install
```
