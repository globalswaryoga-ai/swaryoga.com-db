## üì• Where Fees Are Stored

**File**: `lib/workshopsData.ts`
**Data Structure**: `WORKSHOP_PRICING` object
**Key Information**:
- Workshop slug (unique identifier)
- Duration (days/weeks/months)
- Available modes
- Price per mode
- Optional price label (for special cases)

**Example Structure**:
```typescript
'swar-yoga-basic': {
  durationLabel: '3 days',
  durationDays: 3,
  modes: ['online', 'offline'],
  pricesInrByMode: { online: 96, offline: 96 },
}
```

## üîê Price Updates

To update workshop prices:
1. Edit `WORKSHOP_PRICING` in `lib/workshopsData.ts`
2. Update the specific workshop's `pricesInrByMode` object
3. Restart dev server
4. Prices will automatically update across all pages

**Example**: To change Swar Yoga Basic from ‚Çπ96 to ‚Çπ199:
```typescript

  // ...
  pricesInrByMode: { online: 199, offline: 199 },
}
```
